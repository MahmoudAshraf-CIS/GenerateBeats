# GenerateBeats
Read audio wave and generate beats in place of the sudden freq. change

# install and run 
```
pip install audioowl --no-dependencies
pip install numpy
pip install scip
pip install cython
pip install madmom
pip install librosa
```
make sure python is installed 

open the folder in CMD and run `python script.py`
